🔎 Værktøj · CVR Discovery

Søg i alle aktive danske virksomheder

Direkte lookup mod Datafordeler (CVR-registeret). Brug den når du har en specifik virksomhed i tankerne — eller når du vil eksplorere segmenter manuelt før du sætter en AI-agent på det. Filtrer i højre side, klik for detaljer, og send til Autodialer-køen.

📊 Dashboard

Mål & aktivitet

Dit daglige mål, opkalds-aktivitet, og hvor mange leads der er klar til at ringe. Klargør leads i Leadlist, byg lister i Calling list.

🤖 AI Agent · META Scraper

META Scraper Agent

Agenten kører automatisk to gange dagligt (06:00 + 12:00 CET) og scanner Meta Ad Library for danske virksomheder med aktive annoncer. Nye fund lander automatisk på Dashboard og i Autodialer-køen når de opfylder ICP-reglerne. Juster agentens parametre nedenfor — ændringer træder i kraft ved næste kørsel.

🤖 AI Agent · Maps Scraper

Maps Scraper Agent

Agent der finder danske virksomheder per kategori og by via OpenStreetMap (gratis, ingen API-cost), krydsmatcher dem mod CVR (Datafordeler) for berigelse, og leverer ICP-klare leads direkte til din Autodialer-kø.

🤖 AI Agent · Tech Stack Scanner

Tech Stack Scanner

Agenten finder nye leads ved at gennemgå CVR-puljen, gætte hver virksomheds hjemmeside ud fra navn + brand, probe domænerne, og matche mod tech-signaturer som Shopify, HubSpot, Klaviyo m.fl. Resultater pushes til Dashboard som ny Tech Stack-kilde.

📋 Liste-builder

Byg en målrettet liste fra CVR-registeret (Datafordeler) ved at vælge brancher, by og medarbejderantal. Push hele listen til autodialeren — Apollo enricher og Meta-verify kører automatisk på alle leads.

📋 Leadlist

Workbench

Alle leads i værktøjet — det er her, du klargør dem til opkald. Filtrér efter kilde, branche, eller kvalitet · markér + send til en Calling list · klik en række for at se detaljer.

📞 Calling list

Ring-listen

Dine ring-lister — en pr. niche, branche eller kampagne. Klik "▶ Start opkald" på et kort for at åbne cockpittet. Tilføj nye leads fra Leadlist.

📥 Lead-kilde

Upload list

Upload en CSV-eksport fra Sales Navigator, Apollo, Excel — hvad som helst. Genkendte kolonner: Company / CVR / Website / Phone / Email / City / Industry. Rækker uden CVR-match bliver stadig importeret, flagget som unmatched.

🤖 AI Agent · Look-alike Finder

Look-alike Finder

Agenten lærer fra dine eksisterende leads og finder lignende danske virksomheder i kandidatpuljen — branche, region, størrelse og ads-mønster matches mod hver kandidat. Tilføj 5–10 kvalitetsleads som seed for skarpest output.

🛡 Admin

Oversight

SDR-aktivitet, pipeline health og omkostnings-overblik. Data trukket fra leads-tabellen — server-side credit-tracking kommer i næste runde.

🛡 Admin

Bulk operations

Cleanup + backfill tasks der tidligere kun kunne køres via curl. Hver handling viser hvor mange leads den påvirker før den udfører.
